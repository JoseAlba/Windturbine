Hotwire Calibration

1.Setup the hotwire apparatus.

2.Open up StreamWare.

3.Select new database.

4.Select a suitable name for the project.

5.When asked “no frame /hardware detected" select define a new configuration.

Hotwire Calibration Continued

6.Select the silver "StreamLine Pro" system.

7.Select the appropriate hotwire button. select the appropriate support (normally straight long support are used in lab)

8.Select the wire (4m)

9.Select the temperature probe.

10.Make sure all the wires connected to the wiring board on top of the blue cabinet are same as the channels in the hardware setup.

Hotwire Calibration

11.Click the black “StreamLine” system otherwise the calibration module will not be detected.

12.Click “ok”.

13.After hardware initialization select "overheat adjustment"

14.In the next screen, select "auto-balance"

15.Afterwards, select "operate" and then "play" button

Hotwire Calibration Continued

16.Select the "pause" button, then "stand by" and "close"

17.Select "save hardware event", give an id to it, then "close" the "hardware configuration screen"

18.Right click on the hardware setup in the database list and select "set as default")

19.Select "velocity calibration". Provide the required velocity range and number of test velocities.

20.Check if the nozzle installed is suitable, if yes click "continue"

Hotwire Calibration

21.Select "play" and then "continue"

22.Wait for the calibrator to finish.

23.Click "error analysis". Go through all the point. Acceptable error range is less than 0.5%. If not, perform calibration again.

24.Click "continue", and then close the newly opened "velocity calibration" window and save it.

25.Define an ID, click "yes".

Hotwire Calibration Continued

26.Will ask to define a new "data reduction" setup. Select "yes". Select "yes" and then make it "default"

27.For x probe read step under for three wire prob skip to step 39

28.select "directional calibration"

29.Appropriate nozzle should be already installed. click on "continue with nozzle"

30.Wait for the calibrator to set the velocity.

Hotwire Calibration

31.Press "play" button, select "manually" click "continue".

32.Rotate the calibrator to the indicated angle, click "continue"

33.Repeat for 2nd angle.

34.If the wires are not properly connected, software will indicate the probe is not properly positioned. Click "no" and then switch the wires to hot wire.

35.Repeat for all the angles

Hotwire Calibration Continued

36.Click "error analysis". Go through all the point. The errors here are higher than velocity calibration errors.

37.Click "ok" once satisfactory errors are achieved.

38.Close the newly opened calibration window. Save it, click "ok" to modify the reduction setup.

39.Position the probe so that the three dots are in X-Z plane. Refer to the figure printed on calibrator for axis definitions.

40.Go to step 51 if you just did x-probe probe otherwise continue to do three wire probe

Hotwire Calibration Continued

41.Select directional calibration.

42.Define the inclination angle required. Define the required roll angle range and increment.

43.Appropriate nozzle should be already installed. click on "continue with nozzle".

44.Wait for the calibrator to set the velocity.

45.Press "play" button, select "manually" click "continue".

Hotwire Calibration Continued

46.Set the hotwire at “0” inclination angle. Click “continue”.

47.Set the hotwire to the inclination angle define previously, click “continue”.

48.Set the hotwire at the indicated roll angle, click “continue”.

49.Set the hotwire to the indicated angle and repeat for all angles.

50.After completing the calibration, click “error analysis”. Go through all the point. The errors here are higher than velocity calibration errors.

Hotwire Calibration

51.Go to "setup", select "experiment setup".

52.Select "hardware setup" button in block diagram, check "enable automatic overheat adjustment", click "ok".

53.Select "data flow" in the block diagram, in the new window, select "A/D device" and "temperature/Voltage" buttons.

54.Select the "A/D device" in the block diagram, and change the values of "sampling frequency" and "sample numbers" as required.

55.Click "ok" twice and save the event.

Hotwire Calibration Continued

56.Go to "run", select "disable all".

57.Remove the hotwire from calibration unit, and place it in the wind tunnel.

58.Insert in Transverse system with Hole facing up (Top of Wind Tunnel)

59.Switch on Transverse System (Box on Bottom of CTA Box)

60.After, hooking up all the apparatus in the test section, go to “run” and select “enable all”.

Hotwire Calibration

61.Minimize Streamware

62.Click on Smart Motor Interface

63.Click on Communicate from Menu, Select Address Motor Chain.

64.Wait till ready

65.Click on file open

Hotwire Calibration Continued

66.Select Wakehorizontal

67.File open again, Select the Verticalwake

68.Click Setup

69.Transmit Current Program

70.Transmit Setup

Hotwire Calibration

71.Target Motor, Single Motor 2 is Horizontal and 1 is vertical

72.Transmit now

73.Click Ok

74.Do the same for vertical

75.Click on vertical window change to 1

Hotwire Calibration Continued

76.Transmit now

78.1V=50000

79.1D=140000

80.1G

Hotwire Calibration

81.Streamware Run Enable All

82.Streamware right click Experiment, Run Experiment Minimize program

83.Click on DMA_XFER_1D.exe (on Desktop).

84.#of points in vertical direction: 61 (.1 increments)

85.# of points in horizontal direction:1

Hotwire Calibration Continued

86.Sample Rate: 80,000 Hz

87.Sample # 10,000,000

88.Zone# 1 or 2

89.Then click 1 RSE

90.Save Data in Binary (no)

Hotwire Calibration

91.Then quickly go to Smart motor and type 0RUN in main screen

92.Vv=vertical direction example 61 =6 inches Hh=horizontal direction D=-1000 mean minus because its going down, and means for every 10000 points= 1 inch Sampling rate

93.To gather data, right click on the experiment setup and select "run experiment".

94.To view the raw data for multiple wire probes. Double click the acquired raw data, click “option”

95. To reduce the data, right click on raw data, select “reduce data”.
